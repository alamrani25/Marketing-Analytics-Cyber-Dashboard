# Marketing Analytics Dashboard (Customer Behavior & Campaign Performance)

## 📌 Project Overview
This repository contains a comprehensive **Marketing Analytics Dashboard** developed in Power BI. The dashboard is designed to analyze complex customer datasets, focusing on identifying high-value customer segments, campaign effectiveness, and product-specific sales performance.

The UI is built with a high-contrast dark theme, emphasizing the critical metrics required for data-driven marketing decision-making.

---

## 📊 Dashboard Architecture

### 1. Customer Overview (Page 1)
*   **Spending Insights:** Segmented by Income Group and Age Group.
*   **Demographic Breakdown:** Visualizing customer base by Education level and Marital Status.
*   **Engagement Metrics:** Tracking Response Rate (14.93%) and Average Recency (49.10) to optimize re-engagement strategies.

### 2. Channel & Product Performance (Page 2)
*   **Product Sales:** Detailed performance analysis for categories such as Wines, Meat, and Gold Products.
*   **Campaign Response:** Sum of responses categorized by education, helping identify which segments are most responsive to marketing efforts.
*   **Logistics & Web:** Key indicators like Average Web Visits (5.32) and Complaint Rate (0.89%).

---

## 🛠️ Technical Implementation

*   **ETL Pipeline:** Streamlined data processing using **Power Query** to relate marketing campaign data with customer demographic tables.
*   **DAX (Data Analysis Expressions):** Development of dynamic measures to calculate response rates, average spending, and comparative product performance.
*   **UI/UX:** Specialized "Cyber" theme focused on clarity and aesthetic appeal, ensuring that large datasets remain easily digestible.

---

## 🚀 How to Use
1.  Download the `.pbix` file.
2.  Use the **interactive filters** at the bottom (Year, Income Group, Age Group, Education, Marital Status) to perform real-time segment analysis.
